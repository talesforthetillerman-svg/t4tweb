Layout Debug System Test

Test Instructions

Open these URLs in browser tabs:

  1. /editor?layoutDebug=1
  2. /?layoutDebug=1

In each tab, open DevTools Console (F12) and run:

window.__LAYOUT_DEBUG__.measureAll()

Then export metrics:

window.__LAYOUT_DEBUG__.export()

Copy the JSON output from each tab.

After visiting both routes, you can run:

window.compareHeroLayout()

This will compare the current route with the stored metrics from the other route.
